
Commands Guide:

All bot commands must be preceded with '!'.

!helpme : displays this help information and list of commands
    Variations: !HELP, !commands

!nickname <new_name> : changes nickname in the server to 'new_name'
    !nickname <no input> : resets nickname to original name

!cheer : outputs a random cheer

!ban <user> : bans 'user', only available to permitted moderators/admins

!kick <user> : kicks 'user', only available to permitted moderators/admins
